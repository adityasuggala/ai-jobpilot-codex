# AI Job Pilot Codex

A toolset for automatically tailoring resumes to job descriptions and applying to jobs.

## Components

- **`app/`** - Python modules for resume tailoring, auto-applying, logging, email notifications and a tiny SQLite database.
- **`streamlit_app.py`** - Streamlit web interface to manage applications.
- **`requirements.txt`** - Python dependencies.
- **`.env.example`** - Example environment variables.

This project provides starter code for automating job applications using OpenAI, Selenium and other utilities.
